# Copilot Agent 量化交易 AI 参数优化一体化开发需求方案

## 1. 项目目标

开发一套AI驱动的量化策略参数优化系统，实现如下目标：
- 支持将自定义量化策略（语法/表达式/模板等）通过批量回测与AI算法（如遗传算法、贝叶斯优化）自动寻优参数，提升策略胜率与稳健性，降低回撤。
- 用户可在前端自定义策略、参数空间，发起优化任务，查看优化进度与结果，并一键部署最优参数到实盘系统。
- 系统架构在现有 go_binance_futures（后端）和 go_binance_futures_ui（前端）基础上扩展，主要增加AI优化与批量回测能力。

---

## 2. 总体架构设计

```
前端 (go_binance_futures_ui)
    │
    ├── 接口调用/页面管理/参数空间设置/任务进度与结果展示
    │
Python AI优化服务 (新增)
    │
    ├── 参数空间采样、AI智能寻优（Optuna/GA/贝叶斯等）
    ├── 调用Go后端回测API，收集结果
    ├── 任务调度、进度管理、结果存储
    │
Go后端 (go_binance_futures)
    │
    ├── 策略与参数管理
    ├── 回测API（批量+单次）
    ├── 实盘参数同步API
    │
数据库 (已存在)
    └── 优化任务、参数、回测结果、策略配置等
```

## 7. 参考AI优化脚本片段（Python + Optuna）

```python
import optuna, requests

def run_backtest(params):
    # 伪代码：调用Go后端回测API
    payload = {"strategy_id": 1, "params": params, "start_date": "2023-01-01", "end_date": "2023-12-31"}
    r = requests.post('http://go_server/api/backtest', json=payload)
    return r.json()

def objective(trial):
    ma = trial.suggest_int('ma_period', 5, 60)
    rsi = trial.suggest_int('rsi_threshold', 10, 90)
    res = run_backtest({'ma_period': ma, 'rsi_threshold': rsi})
    score = res['return'] / (res['max_drawdown'] + 1e-6)
    return score

study = optuna.create_study(direction='maximize')
study.optimize(objective, n_trials=100)
```
